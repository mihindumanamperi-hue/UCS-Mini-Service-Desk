# UCS Mini Service Desk

## 📌 Overview
This is a simple Support Ticket Management Web Application built using React (Vite).  
It allows users to create, manage, and track support tickets efficiently.

---

## 🚀 Features

### 🎫 Ticket Creation
- Create tickets with:
  - Title
  - Description
  - Category (Hardware, Software, Network)
  - Priority (Low, Medium, High)
  - Reporter

### 📋 Ticket List
- View all created tickets
- Click on a ticket to view details

### 🔍 Search & Filters
- Search tickets by title
- Filter by:
  - Status (Open, In Progress, Closed)
  - Priority (Low, Medium, High)

### 📄 Ticket Details
- View full ticket information
- Update ticket status
- Add notes to tickets

### 📊 Dashboard Summary
- Displays:
  - Total Open tickets
  - In Progress tickets
  - Closed tickets

---

## 🛠️ Tech Stack
- React (Vite)
- JavaScript
- localStorage (for data persistence)
- CSS (basic styling)
